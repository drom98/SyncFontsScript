# SyncFonts

Little script to automatically install fonts from any sync client folder

### Using

Git clone this repository or download as a zip file.

```
git clone https://github.com/drom98/SyncFontsScript.git
```
Execute the .sh file:
```
./SyncFonts.sh
```
